■ 관리자용 버전 license.dat 생성하기
 ▶ 준비 사항
   - private_key.pem 파일이 있는 path 위치
   - license.dat 파일이 생성될 path 지정이 필요합니다.
   - 고객으로부터 받은 MotherBoard UUID를 입력
   - 고객 CPU Processor ID 입력
 ▶ license.dat 생성 및 검증
   - 준비사항 완료 시, 클릭해서 생성
   - 생성된 파일을 인증 테스트
■ 고객용 버전 사용하기
 ▶ 고객용 설치 파일로 프로그램을 설치 및 실행
   - 라이센스 파일(license.dat) 생성이 필요합니다.
   - 이때 두가지 정보를 받아야 합니다.
   - MotherBoard UUID, CPU Processor ID
   - Get 버튼은 고객PC의 정보를 가져옵니다.
   - 클립보드에 복사 버튼은 각각의 정보를 복사합니다.
   - 이 정보들을 개발사에 전달합니다.
 ▶ license.dat 수령
   - 개발사에서는 위정보기반으로 license.dat 생성
   - 전달받는 license.dat파일을 아래 위치에 복사합니다.
   - C:\TheMR\MifareTool\Bin
   - 인증파일이 문제없으면, 우측 화면이 표시되지 않고 프로그램이 실행됨.

■ 라이센스 인증용 USB메모리 생성 기능 구현
 ▶ 설정 화면에 구현
 1) 라이센스 USB 메모리 생성
 (1) 특정 USB에 연계해서 license.dat 파일을 생성
 (2) 그 USB 루트디렉토리에 license.dat 파일을 복사 후, 배포
 
 2) 배포된 라이센스 USB를 이용, 프로그램 사용
 (1) 배포된 USB메모리를 PC에 장착
 (2) 프로그램 실행 시, USB메모리에 license.dat파일을 인증한다.

 ※ private_key.pem, license.dat파일들의 폴더 위치도 변경 시, 기억한다.
